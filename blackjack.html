Blackjack

(Browser Game)

Project Details

Tools: HTML, CSS, JS

Description: A static site where you can bet chips and play blackjack with all the same rules and logic that you would find in the game at a traditional American casino. HTML elements are all rendered on a "stage" at page load using custom JavaScript functions.

Challenges/Obstacles: One of the rules in blackjack is that the player, upon starting a hand with two cards of the same value, may choose to "split" their hand into an additional hand, with an additional bet, and play each of those hands simultaneously. In order to ensure that game events would reference the correct hand when they occurred, I ended up relying on the path property of the event object, which I only later realized was not supported in Firefox! To make the game work across all browsers, I had to do a little research and fallback onto the standard method event.composedPath().

What I Learned: Working on this project helped give me a lot of practice with conditional logic and event listeners. In retrospect, I think building this project would have been much easier and faster using a JavaScript framework like React, but I also felt that creating a game like this without relying on any frameworks or libraries was a great way to solidify my understanding of the fundamentals.
